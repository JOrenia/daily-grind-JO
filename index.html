At LargeBuck$ we aspire to: Provide you with imaculate unrealistic drinks to view and get you hungry enough to actually got to go to a real coffee shop. This page is uber cool but also uber fake. Happy coffee drinking!

In this assignment, we created a webpage for a coffee shop that dynamically changes its page contentts depending on what day it is. I created a Js folder which holds the javaScript functions and switches for each day. In the JavaScript file, I added unique content such as image, day of the week, description, alt attribute, name of the drink, and different page color for each day. using a single JavaScript page, I can rotate between the unique contents for each day using the Date objects and also the swtich block method to populate the correct page for each day.

The main page that load will show the day and the correspnding drink with its unique contents. In order for users to view different days, I created links for each day that references the same index.html file but using the query string to call different days of te week.
